= Flusso di lavoro di installazione del software per gli switch di archiviazione Cisco Nexus 3232C
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Per installare e configurare il software per uno switch Cisco Nexus 3232C e installare o aggiornare il file di configurazione di riferimento (RCF), attenersi alla seguente procedura:

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Uno"]link:setup-switch-3232c-storage.html["Configurare l'interruttore"]
[role="quick-margin-para"]
Configura lo switch di storage 3232C.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Due"]link:prepare-install-cisco-nexus-3232c-storage.html["Prepararsi all'installazione del software NX-OS e RCF"]
[role="quick-margin-para"]
Il software Cisco NX-OS e i file di configurazione di riferimento (RCF) devono essere installati sugli switch di archiviazione Cisco 3232C.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Tre"]link:install-nx-os-software-3232c-storage.html["Installa o aggiorna il software NX-OS"]
[role="quick-margin-para"]
Scarica e installa o aggiorna il software NX-OS sullo switch di archiviazione Cisco 3232C.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quattro"]link:install-rcf-software-3232c-storage.html["Installare l'RCF"]
[role="quick-margin-para"]
Installare l'RCF dopo aver configurato per la prima volta lo switch di archiviazione Cisco 3232C.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinque"]link:upgrade-rcf-software-3232c-storage.html["Aggiorna l'RCF"]
[role="quick-margin-para"]
Aggiorna la versione RCF esistente se necessario.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["Sei"]link:configure-ssh-keys.html["Verifica la configurazione SSH"]
[role="quick-margin-para"]
Verificare che SSH sia abilitato sugli switch per utilizzare le funzionalit√† di monitoraggio dello stato dello switch Ethernet (CSHM) e di raccolta dei registri.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-7.png["Sette"]link:reset-switch-3232c-storage.html["Ripristinare l'interruttore alle impostazioni predefinite di fabbrica"]
[role="quick-margin-para"]
Cancella le impostazioni dello switch di storage 3232C.
