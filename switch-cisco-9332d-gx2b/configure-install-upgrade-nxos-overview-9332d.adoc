= Installa o aggiorna la panoramica di NX-OS - Cisco Nexus 9332D-GX2B
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puoi installare o aggiornare NX-OS sugli switch Nexus 9332D-GX2B per i sistemi AFX 1K e AFX 2K.


NOTE: L'installazione o l'aggiornamento di NX-OS è indipendente dall'installazione o dall'aggiornamento di RCF. Se stai installando o aggiornando sia NX-OS che RCF, ti consiglio di installare o aggiornare prima NX-OS.

* link:configure-install-nxos-9332d.html["Installa NX-OS"]
+
Installi NX-OS sugli switch come parte dell'installazione di un nuovo sistema AFX 1K o AFX 2K, della sostituzione di uno switch o del ripristino di uno switch alle impostazioni di fabbrica.

* link:configure-upgrade-nxos-9332d.html["Aggiorna NX-OS"]
+
Aggiorni NX-OS sugli switch per supportare nuovo hardware o una nuova versione di ONTAP sul tuo sistema AFX 1K o AFX 2K.
