== Supporto per porte Ethernet NVIDIA CX6-DX

Se si collega una porta dello switch a un controller ONTAP utilizzando porte NIC NVIDIA ConnectX-6 Dx (CX6-DX) (codice parte X50130B), è necessario codificare la velocità della porta dello switch.

[listing, subs="+quotes"]
----
(cs1)# *configure*
(cs1)(config)# *interface Ethernet1/16/?*
(cs1)(config-if)# *speed 100000*
(cs1)(config-if)# *no negotiate auto*
(cs1)(config-if)# *exit*
(cs1)(config)# *exit*
(cs1)# *copy running-config startup-config*
----
Vedere https://hwu.netapp.com/Switch/Index["Hardware Universe"^] per ulteriori informazioni sulle porte dello switch.
