== Archiviazione via cavo NS224 come switch-attached

Se si desidera cablare l'archiviazione NS224 come collegata allo switch, seguire lo schema relativo:

image:9336c_image1.jpg["Interruttore collegato"]

Consultare la link:https://hwu.netapp.com/Switch/Index["Hardware Universe"^] per ulteriori informazioni sulle porte dello switch. Consultare link:https://kb.netapp.com/Support/General_Support/What_additional_information_do_I_need_to_install_my_equipment_that_is_not_in_HWU["Quali informazioni aggiuntive mi servono per installare la mia attrezzatura che non è presente in HWU?"^] per ulteriori informazioni sui requisiti di installazione dello switch.



== Archiviazione via cavo NS224 come collegamento diretto

Se si desidera cablare l'archiviazione NS224 come collegamento diretto anziché utilizzare le porte di archiviazione dello switch condiviso, seguire lo schema del collegamento diretto:

image:9336c_image2.jpg["Attacco diretto"]

Consultare la link:https://hwu.netapp.com/Switch/Index["Hardware Universe"^] per ulteriori informazioni sulle porte degli switch.



== Foglio di lavoro sul cablaggio Cisco Nexus 9336C-FX2

Se si desidera documentare le piattaforme supportate, è necessario compilare il foglio di lavoro di cablaggio vuoto utilizzando come guida il foglio di lavoro di cablaggio di esempio completato.

La definizione di porta di esempio su ciascuna coppia di switch è la seguente:

image:cabling-worksheet.jpg["Foglio di lavoro sul cablaggio"]

Dove:

* 100G ISL per commutare la porta A 35
* 100G ISL per commutare la porta A 36
* 100G ISL per commutare la porta B 35
* 100G ISL per commutare la porta B 36




== Foglio di lavoro vuoto per il cablaggio

È possibile utilizzare il foglio di lavoro vuoto per il cablaggio per documentare le piattaforme supportate come nodi in un cluster. La tabella Supported Cluster Connections del link:https://hwu.netapp.com["Hardware Universe"^] definisce le porte del cluster utilizzate dalla piattaforma.

image:blank-cabling-worksheet.jpg["Foglio di lavoro vuoto per il cablaggio"]

Dove:

* 100G ISL per commutare la porta A 35
* 100G ISL per commutare la porta A 36
* 100G ISL per commutare la porta B 35
* 100G ISL per commutare la porta B 36
