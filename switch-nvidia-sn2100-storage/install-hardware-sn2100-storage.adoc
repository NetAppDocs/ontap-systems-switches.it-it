= Installare l'hardware per lo switch NVIDIA SN2100
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
L'installazione dell'hardware dello switch NVIDIA SN2100 è un primo passo fondamentale per creare un'infrastruttura di rete affidabile e scalabile a supporto dei flussi di lavoro di protezione, replica e gestione dei dati.

Segui questi passaggi per installare e configurare gli switch SN2100. Consulta la documentazione di NVIDIA per istruzioni dettagliate.

.Passi
. Rivedere illink:configure-reqs-sn2100-storage.html["requisiti di configurazione"] .
. Seguire le istruzioni nel link:https://docs.nvidia.com/networking/display/sn2000pub/Installation["Guida all'installazione dello switch NVIDIA"^].


.Cosa succederà ora?
Dopo aver installato l'hardware, puoilink:cabling-considerations-sn2100-storage.html["rivedere il cablaggio e la configurazione"] requisiti.
